# projet-boutique


##  Description

Application web e-commerce développée avec PHP, MySQL, HTML et CSS selon le modèle MVC.

##  Installation

1. Installer XAMPP
2. Cloner le projet :

   ```
   git clone https://github.com/username/projet-boutique
   ```
3. Placer le dossier dans :

   ```
   C:/xampp/htdocs/
   ```

##  Base de données

1. Ouvrir phpMyAdmin
2. Créer une base :

   ```
   projet_boutique
   ```
3. Importer le fichier :

   ```
   database.sql
   ```

##  Lancer le projet

* Démarrer Apache + MySQL
* Aller sur :

  ```
  http://localhost/projet-boutique
  ```

##  Fonctionnalités

* Inscription utilisateur
* Connexion / Déconnexion
* Panier
* Recherche produits
* Gestion commandes

##  Technologies

* PHP (POO + MVC)
* MySQL (PDO)
* HTML / CSS
* XAMPP
